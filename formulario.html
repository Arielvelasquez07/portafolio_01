Proyecto 3: Formulario

Descripción del Proyecto

Este proyecto consiste en la creación de un formulario web interactivo. El formulario permite a los usuarios enviar información de manera fácil y eficiente, con validación en tiempo real para asegurar que los datos ingresados sean correctos y completos.

Tecnologías Utilizadas

  • HTML5
  • CSS
  • JavaScript
  • Bootstrap 5

Características

  • Diseño responsivo
  • Validación en tiempo real
  • Interfaz amigable y fácil de usar
